Bienvenido a Ing. Web Project

Desarrollado por el GRUPO 03

IWP (Ing. Web Project)


El presente proyecto surge como idea del Ing. Marcelino Torres para el curso de Ingeniería Web dictado a los alumnos de 9eno ciclo de la Universidad Nacional de Trujillo.

Tecnologías

HTML

HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web basándose en etiquetas.

CSS

Cascading Style Sheets es un lenguaje de diseño gráfico para definir la presentación de un documento estructurado escrito en un lenguaje de marcado.

JavaScript

Es un lenguaje de programación interpretado, se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Bootstrap

Es una biblioteca de código abierto para diseño de sitios y aplicaciones web. Contiene plantillas de diseño con elementos de diseño basado en HTML, CSS y JS.

Spring Framework

Es un framework para el desarrollo de aplicaciones y contenedor de inversión de control, de código abierto para la plataforma Java.

MySQL

Es un sistema de gestión de bases de datos relacional considerada como la base de datos de código abierto más popular del mundo.

Tablas Principales

Roles

Se almacena todos los roles del sistema

Usuarios

Se almacena todos los usuarios del sistema, con su respectivo rol

Formatos

Se almacena todos los formatos de video. Por ejemplo: MP4, AVI, etc.

Generos

Se almacena todos los generos de video. Por ejemplo: TERROR, ACCION, etc.

Idiomas

Se almacena todos los idiomas de video. Por ejemplo: INGLES, CASTELLANO, etc.

Urbanizaciones

Se almacena todos las urbanizaciones. Por ejemplo: QUINTANAS, TRUJILLO, etc.

Clientes

Se almacena todos los clientes relacionado a una urbanizacion

Videos

Se almacena todos los videos relacionado a un formato, varias idiomas y generos

Fichas

Se almacena todo el proceso del alquiler de video.
